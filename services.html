Common questions

Do I own the code?

Yes, 100%. Full source code transfer is included in every package. No licensing fees, no lock-in.

What hosting do I need?

A VPS with 8GB RAM runs the full stack. I recommend Hetzner (~€6/mo) or DigitalOcean (~€48/mo). Setup is included in every package.

Can you integrate with Shopify or existing tools?

Yes. The API Gateway exposes standard REST endpoints that can connect to any third-party service, ERP, or payment provider.

How long does it take?

Starter: 1–2 weeks. Custom: 3–5 weeks. Full Build: 6–10 weeks, depending on feature scope.
